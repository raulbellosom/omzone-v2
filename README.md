# OMZONE AI Kit

Bundle base de agentes, prompts, instructions y skills para usar con Copilot, Claude, Códex o herramientas similares.

## Contenido
- `.github/agents/`
- `.github/instructions/`
- `.github/prompts/`
- `.github/skills/`
- `.github/copilot-instructions.md`

## Uso sugerido
1. Copiar al repositorio del proyecto
2. Ajustar herramientas soportadas por tu editor/CLI
3. Enlazar con `docs/core/00_documento_maestro_requerimientos.md`
4. Refinar agentes según el stack exacto del frontend
